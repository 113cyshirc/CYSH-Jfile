# CYSH Judge 檔案系統
# 檔案類型
## .cjz - CYSHJ題目檔
在進行傳輸時使用，單檔案，可透過我們特製的系統import。

## .cjf - CYSHJ描述檔
題目的描述、設定

## .cjt - CYSHJ測資檔
測資的檔案，將以往的in、out檔合併的新檔案模式。
